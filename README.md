Project ini adalah tugas praktikum untuk menerapkan CameraX di Android Studio menggunakan Jetpack Compose. Fungsinya sederhana: menampilkan preview kamera, mengambil foto, dan menyimpannya ke MediaStore.
Fitur
1. Preview kamera real-time
2. Tombol Capture untuk mengambil foto
3. Foto otomatis tersimpan di: Pictures/CameraX-Demo/
4. Tombol Switch untuk pindah kamera depan/belakang
5. Tombol Flash (torch mode) untuk menyalakan/mematikan lampu
6. Permission kamera menggunakan runtime permission Android
